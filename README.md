### Features:

- Responsive design
- Fast loading due to Webpack, Babel, PostCSS minification and the absence of frameworks. Configured from scratch

### Stack:

- Pure JavaScript for slider and menu
- Webpack
- Babel
- PostCSS
- SCSS
- HTML

### Use:

Install: `npm install`
Build: `npm run build`
Dev server: `npm start`
